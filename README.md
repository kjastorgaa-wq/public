# Dashboard de Análisis de Demandas (Caso Práctico)

Este proyecto consiste en el desarrollo de un tablero interactivo en Power BI para analizar un conjunto de datos sobre denuncias de defensa del consumidor. El objetivo es identificar los sectores con mayor volumen de quejas, tiempos de resolución y patrones de comportamiento.

## 📊 Contenido del Repositorio
* **denuncias-defense-del-consumidor.csv**: El conjunto de datos original utilizado para el análisis.
* **Informe de Denuncas.pbix**: El archivo de Power BI con el modelo de datos y el reporte visual.
* **Ejercicio Práctico Power Bi.pbix**: Versión de práctica y desarrollo del modelo.
* **Panel/**: Carpeta con los recursos visuales e imágenes de fondo utilizadas para la interfaz.

## 🛠️ Herramientas y Tecnologías Utilizadas
* **Power BI**: Para el modelado de datos, creación de medidas en DAX y diseño de tableros.
* **Power Query**: Para la limpieza, transformación y preparación del archivo CSV.
